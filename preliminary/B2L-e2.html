Listening Comprehension, Part 2


You will hear a radio interview. First you will have one minute to read the introduction and the items. Then you will hear the interview. You will hear the interview twice.

Decide if the statement for each of the items is true or false.

Now you will have one minute to read the items.

  1. 01

    The captain tried to manoeuvre the ship around the reef.

    1. True
    2. False
  2. 02

    Crews can contain the slick if it is reached quickly enough.

    1. True
    2. False
  3. 03

    Environmentalists support the use of chemicals in order to rescue wildlife.

    1. True
    2. False
  4. 04

    Pelicans can be found all year round in Callo Bay.

    1. True
    2. False
  5. 05

    Rescue workers tranquilize the birds before treating them.

    1. True
    2. False
  6. 06

    Household cleaning products are generally used to remove oil from birds.

    1. True
    2. False
  7. 07

    The Alaskan rescue effort was unsuccessful.

    1. True
    2. False
  8. 08

    Wildlife relocation is a common response to an oil spill.

    1. True
    2. False
  9. 09

    The volunteers were drawn from the immediate surroundings of the disaster.

    1. True
    2. False
  10. 10

    The methods used in Chile are only suitable for that region.

    1. True
    2. False
